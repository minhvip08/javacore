# Learning Docs - Simi

Welcome to Simi's learning documentation repository! This workspace is dedicated to tracking progress, notes, and code across various technologies and frameworks.

## Repository Structure

This workspace is divided into several main learning modules:

### 1. Java Core (`/java`)
Contains foundational and advanced Java concepts, structured by sections (Section 3 through 30). 
- Covers deep dives into core Java modules, syntax, and advanced programming topics.

### 2. gRPC Java (`/grpc-java`)
Exploration of gRPC features and implementations in Java.
- Includes building APIs, client-server definitions, and using protocol buffers.
- See the specific `grpc-java` README for detailed build and run instructions.

### 3. Apache Thrift (`/thrift`)
Projects and examples using Apache Thrift for scalable cross-language services development.
- Includes build scripts and documentation for creating Thrift services.

## Getting Started

To get started, you can clone this repository to your local machine.

### Cloning this repository
```bash
git clone <your-repository-url>
cd javacore
```

### Cloning gRPC Java
If you need to clone the official `grpc-java` repository separately for reference or updates, you can run:
```bash
git clone https://github.com/grpc/grpc-java.git
```

Feel free to browse through the directories above. Each technology stack contains its own set of instructions and examples.
